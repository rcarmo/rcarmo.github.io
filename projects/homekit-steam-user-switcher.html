About

A household with multiple Steam accounts on a shared PC meant walking to the TV every time someone wanted to play. This Python service uses HAP-python to expose each Steam account as a HomeKit Switch. Tap it in the Home app, tell Siri, or use it in an automation — it logs out the current user and logs in the target account via Steam's local CLI flags. Runs as a systemd service.

Features
What it does
🏠
HomeKit native
Each Steam account is a HomeKit Switch. Control it from the Home app, Siri, Shortcuts, or a time-based automation.
🎮
No Steam API required
Uses Steam's --login flag locally. No Steam Web API account, no OAuth flow, no third-party services.
🐧
macOS and Linux
HAP-python accessory server. Runs as a systemd service on the gaming machine itself.
