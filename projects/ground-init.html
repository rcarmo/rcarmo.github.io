About

cloud-init is overkill for machines you configure by hand. ground-init is a Python script that takes a YAML config and idempotently applies it to a fresh Ubuntu, Debian, or Fedora system — packages, users, SSH keys, dotfiles, systemd services — with clear output at every step.

Features
What it does
📄
YAML-driven
Declare the desired state in a YAML file. ground-init applies it, reports what changed, and does nothing on re-runs when state is already correct.
🔄
Idempotent
Run it once to set up, run it again to verify. Each step reports explicitly: changed, skipped, or failed.
🐧
Multi-distro
Ubuntu, Debian, Fedora, and Fedora Silverblue (rpm-ostree). Ships with real-world sample configs as starting points.
📦
Zero dependencies
Pure Python 3 stdlib. No Ansible, no Chef, no pip install — just Python and a text editor.
