About

Vibes is a lightweight web frontend that works with both ACP (GitHub Copilot, OpenAI Codex) and Pi agents via RPC. Built for the phone screen first: nothing to install, no build step, just a clean interface for talking to your agent from anywhere.

Features
What it does
📱
Mobile-first
Designed for real use on a phone. The layout, input handling, and scroll behaviour are tested on iOS and Android before desktop.
⚡
ACP + Pi
Works with `copilot --acp`, `codex-acp`, and Pi agents via RPC. Protocol support is pluggable — add a new adapter in one file.
🚿
Streaming responses
Server-sent events for live token streaming. No polling, no page refreshes, no spinners.
🔩
Zero build step
A single Python file for the server and a single HTML file for the UI. Fork it and make it yours in an afternoon.
