About

Vibes is a lightweight Python web app for talking to AI coding agents from a phone. It supports the Agent Communication Protocol (copilot --acp, codex-acp) and Pi agents via RPC. The server is a single Python file using aiohttp with server-sent events for streaming. The UI is plain HTML/CSS — no build step, no framework. Fork it and make it yours in an afternoon.

Features
What it does
📱
Phone-first layout
Designed on a phone, tested on iOS and Android before desktop. Layout, scroll, and input handling all work correctly on a small screen.
⚡
ACP + Pi RPC
Connects to GitHub Copilot CLI, OpenAI Codex, and Pi agents. Protocol adapters are pluggable — add a new backend in one file.
🚿
SSE streaming
Live token streaming via server-sent events. No polling, no loading spinners, no page refreshes.
🔩
Zero build step
One Python file for the server, one HTML file for the UI. Clone and run. PiClaw and Vibes share the same web UI codebase.
