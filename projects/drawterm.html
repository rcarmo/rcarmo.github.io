About

Standard drawterm renders at 1x on a 2x Retina screen, making Plan 9 and 9front sessions look like fine print. This fork patches the macOS rendering path to read the display scale factor and apply it — two lines of code that make a real difference.

Features
What it does
🔍
HIDPI scaling
Reads the macOS display scale factor and applies it to drawterm's rendering. Retina and 5K displays work correctly instead of rendering at half size.
🍎
Apple Silicon native
Compiles natively for arm64. No Rosetta, no emulation layer.
🧬
Minimal diff
The smallest possible change from upstream drawterm — easy to rebase when upstream updates.
