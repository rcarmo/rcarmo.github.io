About

A maintained fork of the Basilisk II (68k) and SheepShaver (PowerPC) Macintosh emulators with fixes for building on macOS 14+ and modern Linux. Networking via slirp means no tun/tap, no root required. Used as the emulation layer for the Maclock and mini Mac projects.

Features
What it does
💻
68k + PowerPC
Basilisk II for System 6–8 (68k Macs). SheepShaver for Mac OS 8.6–9.0.4 (PowerPC). Both build cleanly from the same repo.
🌐
Networking via slirp
NAT networking without tun/tap or root privileges. Access the internet from inside the emulator on any host platform.
🔧
Modern build
CMake build system. Fixes for macOS 14+ and glibc 2.38+ compile issues. No patched headers, no manual SDK tweaks.
📟
Used in Maclock
The emulation engine running inside the Maclock project — a $20 alarm-clock-shaped Mac replica repurposed as a proper Mac Classic.
