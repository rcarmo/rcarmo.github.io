About

This fork of the Basilisk II (68k Mac) and SheepShaver (PowerPC Mac) emulators targets Raspberry Pi with SDL2 framebuffer/KMS rendering — no X11 or desktop required. Pre-built .deb packages for arm64 are available in releases. Build scripts use an optimised SDL2 build that runs on the Pi's bare console, making it practical to run classic Mac software on a headless Raspberry Pi.

Features
What it does
🥧
Raspberry Pi optimised
SDL2 built with framebuffer/KMS, no OpenGL, no X11. Runs on a headless Pi Zero through Pi 5 — no desktop environment needed.
📦
Pre-built packages
arm64 .deb packages available from GitHub Releases. dpkg -i and you're done. Build from source instructions are included for other targets.
💻
68k and PowerPC
Basilisk II runs System 6–8 on 68k. SheepShaver runs Mac OS 8.6–9.0.4 on PowerPC. Both emulators share the same build system.
🖥
Direct framebuffer access
SDL2 framebuffer backend writes directly to the display buffer — lower latency and less overhead than running through X11 or a compositor.
