About

go-te is the shared terminal state engine used by webterm, go-rdp, and ghostty-web. It parses VT100/VT220/xterm escape sequences, maintains a cell buffer with full attribute support, and fires update callbacks — giving applications a correct terminal layer they can render however they like.

Features
What it does
🔣
Full VT100/xterm support
Escape sequences, CSI commands, colour attributes, alternate screen buffer, and cursor movement — the complete set needed for real-world terminal applications.
📦
Embeddable
A Go library, not a server. Import it, provide a callback for screen updates, and render the cell buffer however your application needs.
⚡
Zero CGO
Pure Go. Compiles to a static binary for any GOARCH. No libvte, no ncurses, no system library dependencies.
