About

The Totem is one of the most compact practical split keyboards but its ZMK config ecosystem is scattered. This config adds ZMK Studio support for GUI-based keymap editing, home row mods with tuned timing, and combos that make 34 keys actually comfortable for daily writing and coding.

Features
What it does
🖥
ZMK Studio
GUI keymap editing without reflashing. Adjust layers, combos, and key assignments from a web interface — no toolchain required.
✋
Home row mods
Mod-tap on the home row with per-key timing tuned to minimise accidental triggers during fast typing.
⌨
34-key layout
Combos for common brackets and symbols make all punctuation reachable without a dedicated symbol layer.
🍴
Used as a template
More forks than stars — this config is primarily used as a starting point for personal Totem keymaps.
